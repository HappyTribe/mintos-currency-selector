# Mintos Currency Selector

This is an test application for requesting international exchange rates.

## Installation

First clone the repository to your local machine using git commands and go into the folder

```bash
git clone <REPO_URL> mintos-currency-selector
cd mintos-currency-selector
```

Use the package manager [npm](https://www.npmjs.com/) to install the required dependencies

```bash
npm install
```

## Usage

To start the project in development mode type

```bash
npm run start
```

To launch the projects test suites type

```bash
npm run test
```
